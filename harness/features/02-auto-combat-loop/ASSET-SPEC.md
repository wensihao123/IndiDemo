## 2. Naming & format(命名与格式 —— 先锁,勿后改)
- 文件名全小写 `snake_case`,见上表(扩展名 `.png`)。
- 目录:敌人 `assets/sprites/enemies/`、FX `assets/sprites/fx/`、战士 `assets/sprites/hero/`、
  背景 `assets/sprites/bg/`、handle `assets/sprites/ui/`。
- 格式 **PNG**;除 `bg_strip`(满底不透明)外全部透明背景。
- 源文件(.kra/.psd 等)放 `res://_source/`(带 `.gdignore`,不导入/不进导出包)。
- **命名一旦被场景引用就别改**(改名级联断引用,style-basic-2d §5.2)。

## 3. Style constraints(外观约束 —— 全部回扣 STYLE-BIBLE 方案 A)
- **画风**:平滑手绘、**非像素**;圆润 Q 萌、头身比 1:1.3~1:1.5;明度偏高、低饱和(§3)。
- **描边**:统一 `#3a2e2a` 暖棕(非纯黑),主轮廓 ~3px;**单层柔和 cel 阴影**(主色 ×0.85),
  高光极简(§3)。
- **配色落点**(§4):
  - 哥布林系(1/4):清新绿 `#7fb069` 体 + 深绿 `#4a7c59` 暗面;哥布林王加暖橙 `#e8943c` 王冠点睛。
  - 兽人系(3/5/8):暖木 `#c98a5e` 偏褐肤;酋长加暖橙强调(图腾/披风)区分 Boss。
  - 狼系(2/6):野狼用暖木/奶白灰褐;暗影狼压暗、偏深紫灰 `#2b2536` 调,但**仍非纯黑、仍要可爱不恐怖**。
  - 食人魔(7):大体型暖木 + 深绿,憨而非血腥。
  - **Boss 共性**:体型更大 + 一处暖橙 `#e8943c` 点睛(王冠/图腾/披风),让玩家一眼读出"这是关口门"。
- **可读性(支柱 1 核心验收)**:剪影要能在 ~70–125px 显示高、且余光一瞥就分得清"这是哪种怪 / 是不是 Boss";
  同系怪(两只兽人、两只狼)靠装备/明暗/体型区分,不靠细节。
- **基调守支柱 1**:即便是怪也走"可爱治愈"而非写实凶残——它是陪伴向挂机游戏,不要血腥/惊悚。
- **FX**:`fx_light_pillar` 中性白、上窄下宽/柔边、顶部渐隐,**不带颜色**(运行时染);
  `fx_loot_sparkle` 柔和放射星辉、中性白、边缘透明。**FX 里不得出现敌人/背景/UI/数字**(§6.3 只表达事件)。
- **不得包含(运行时状态,style-basic-2d §3)**:血条、HP 数字、伤害飘字、选中框、范围圈、
  受击闪白/中毒冰冻染色、等级数字、朝向第二套(用 `flip_h`)。

## 4. Prompt brief(每件的生成"意图" —— 交 image-prompt 技能编译成品 prompt)
> 以下是**意图 brief,不是成品 prompt**。成品由 `image-prompt` 技能套项目四段前缀编译。
> 共同锚点:平滑手绘非像素、`#3a2e2a` 暖棕描边、方案 A 暖调高明度低饱和、圆润 Q 萌、侧视、透明背景、单张静态。

**敌人共同 brief**:类 MapleStory 的 Q 萌侧视小怪,**面朝左**,全身站/伏在地面、脚底贴画布底居中,
中性待机姿(无攻击动作、无被击),可爱不凶残;**不要**血条/数字/特效/背景/朝向第二套。
- **#1 enemy_goblin**:瘦小绿皮哥布林,大耳尖牙小个子,手持破木棒;最弱杂兵感,清新绿为主。
- **#2 enemy_wolf**:四足灰褐野狼,体型中等,警觉姿;暖木/奶白灰褐,蓬松圆润不写实。
- **#3 enemy_orc**:壮实绿褐兽人,比哥布林大一圈,握粗陋斧;暖木偏褐肤 + 深绿暗面。
- **#4 enemy_goblin_king(Boss)**:更大更壮的哥布林,**戴暖橙王冠**、披小斗篷,气场强;一眼是"头目"。
- **#5 enemy_orc_elite**:精英兽人,比普通兽人多一层简甲/肩甲,神态更凶悍但仍 Q;与 #3 靠护甲区分。
- **#6 enemy_wolf_shadow**:暗影狼,压暗偏深紫灰、毛尖透微光的"影"感,仍圆润可爱不惊悚;与 #2 靠明暗区分。
- **#7 enemy_ogre**:巨型食人魔,大肚憨壮、握巨棒,体型明显最大(普通怪里);暖木 + 深绿,憨而非血腥。
- **#8 enemy_orc_chieftain(Boss)**:兽人酋长,最魁梧,持**暖橙图腾杖/披兽皮**,首领威仪;关 2 终门担当。

**FX brief**:
- **#9 fx_light_pillar**:一道柔和竖直光束,**中性白**、底宽顶窄、边缘柔化、顶端渐隐透明,
  从地面升起的"掉落提示光柱"意象;**纯效果、无任何主体/背景/颜色**(颜色运行时染蓝或金)。
- **#10 fx_loot_sparkle**:一小团柔和放射状星辉/光点迸发,**中性白**、居中、四周透明;
  作掉落落点的"惊喜"微光(金掉落染金更亮);**无主体/无背景/无文字**。

**01 转正 brief**:
- **#11 hero_warrior**:类 MapleStory Q 萌战士,**面朝右**,持剑与小盾,暖调高明度、中性待机站姿,
  脚底贴画布底居中;身上避免左右不对称且镜像后会出错的标志(要能 `flip_h` 复用)。
- **#12 bg_strip**:柔和手绘氛围背景条,暖木城镇/户外清新绿基调,低对比低信息量,**左右无缝平铺**;
  **无角色/UI/文字/玩法实体**。
- **#13 icon_handle**:极简 Q 萌徽记(小队/盾牌意象),强轮廓、小尺寸可读、透明背景、**不烘焙文字**。
